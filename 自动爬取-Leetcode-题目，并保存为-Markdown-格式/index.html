原理

leetcode 的请求都是用的 GraphQL 技术。相比起 RESTful 格式而言，它更加灵活，能够减少请求数（一次就能查询多个数据）。查询的时候可以只请求部分内容。

对于本例而言，用 F12 ，然后看 Network，搜索graphql的请求后，筛选一下就能得到合适的结果
